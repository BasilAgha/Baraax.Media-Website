# Baraax Media

Premium minimalist cinematic media-agency website built with React, Vite, TailwindCSS, and Framer Motion.

## Scripts

```bash
npm install
npm run dev
```

## Structure

- `src/App.tsx` - Single-page website composition
- `src/components` - Reusable sections and UI components
- `src/content` - Category and portfolio content model
- `public/media` - Starter media placeholder notes
